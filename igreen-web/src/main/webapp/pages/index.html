公司简介

绿融（北京）投资服务有限公司（简称“绿融”，英文名“the iGreenBank”）是中国首家绿色金融科技服务商。基于对金融、环境、能源、气象等领域的深刻理解与相关领域大数据的深厚积累，依托自主研发的AI核心技术，绿融开发了EnRobot 1.0绿色金融人工智能引擎。 EnRobot已支持包括绿色债券、绿色指数、绿色基金等总计近1000亿元人民币的绿色金融产品开发和应用，服务于近50家各类金融机构和各级政府等。iGreenbank已经成为中国最顶尖的绿色金融科技企业。

推出EnRobot绿色金融人工智能引擎

2017年3月

研发的国内首个环境领域AI系统“EnRobot1.0”人工智能引擎投入实际应用，已支持近50家金融机构及各级政府的绿色金融产品开发和业务推进。

合作研发中国首个绿色股票指数

2017年5月

绿融与中证指数公司、南方基金共同合作研发编制的中国第一个绿色股票指数——中证绿色投资股票指数（指数代码:930956）正式发布。

担任世界银行技术顾问

2017年3月

由绿融与中国标准化研究院联合承担的世界银行中国节能融资项目，帮助民生银行开发合同能源管理融资业务，结合商业银行的产品特点、风险偏好、内控体系等提出可行的产品设计方案。该项目成果获得了世界银行、民生银行及课题组专家的高度评价。

承担多项地区绿色金融规划

2016年12月

主持或参与了北京市、广州市花都区、浙江省衢州市、山西省大同市、青海省西宁市、前海自贸区和南沙自贸区等地的绿色金融发展规划与实施方案编制工作，帮助地方政府搭建绿色金融服务平台。
